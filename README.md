# The Community Wall

A global digital mosaic built by creators everywhere.  
Upload a drawing → mint it as an NFT → your art becomes a permanent tile on a shared community wall.

---

## Overview

The Community Wall is a collaborative art experience powered by blockchain.  
Anyone can upload a drawing, mint it as an NFT, and contribute to a massive mosaic created by people around the world.

This project was built during a hackathon to explore how creativity, community, and Web3 can intersect in a fun and meaningful way.

---

## Features

### • Upload Your Tile
Users can upload any drawing or image, which is instantly previewed before minting.

### • NFT Minting (Sepolia Testnet)
Every image becomes a unique NFT stored permanently on the blockchain.

### • Global Mosaic
All minted tiles appear on the shared “Community Wall,” forming a massive crowd-built artwork.

### • Clean, Modern UI
Glassmorphism, smooth animations, responsive layout, and a polished interactive flow.

### • Fully On-Chain Ownership
Minted NFTs are owned by their creator and visible to anyone.
